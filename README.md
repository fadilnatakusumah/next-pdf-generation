## Next PDF Converter

First, you need to put your Browserless API token inside the `.env.local`
```bash
BROWSERLESS_TOKEN=
```





Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser and use the app.
